## Usage
```
git clone https://github.com/AlexanderAjagba/leet-analyzer.git
```
1. Right-click (or Control-click) LeetTrack.app
2.	Click Open
3.	Click Open again in the dialog

(System Settings)
- Go to System Settings → Privacy & Security → Scroll down to Security → Click Open Anyway next to LeetTrack → Confirm
